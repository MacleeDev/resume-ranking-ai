# Resume AI Analyzer

**A Smart AI-Powered Resume Screening System (ATS Killer)**

---

## Overview

Resume AI Analyzer is a web-based application that helps job seekers **optimize their resumes** for specific job descriptions.  
It uses AI and NLP techniques to **analyze resumes, compare them to job requirements, and provide actionable suggestions** to improve match scores.

This project is perfect for demonstrating your skills in **Python, FastAPI, NLP, and frontend development**, making it a strong addition to any portfolio.

---

## Features

- **Upload a Resume (PDF)** – Drag & drop support.
- **Paste a Job Description** – Easy input for job requirements.
- **AI-Powered Analysis**:
  - Calculates **match score** between resume and job description.
  - Highlights **matched skills** and **missing skills**.
  - Provides **suggestions** to improve your resume.
- **Interactive Frontend**:
  - Match score **progress bar**.
  - Matched skills highlighted in **green**, missing skills in **red**.
  - Optional **downloadable PDF report**.
- **Tech Stack**:
  - Backend: **FastAPI**
  - AI/NLP: **sentence-transformers**, **scikit-learn**
  - PDF Parsing: **PyPDF2** / **pdfplumber**
  - Frontend: **HTML, CSS, JavaScript**
  - Optional upgrade: **React**
