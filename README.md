# Resume AI Analyzer

AI-powered system to analyze resumes against job descriptions.

## Features
- Upload PDF resumes
- Analyze against job descriptions
- Shows match score, matched skills, missing skills
- Suggestions to improve resume

## Tech Stack
- Backend: FastAPI
- AI/NLP: sentence-transformers, scikit-learn
- PDF Parsing: PyPDF2 / pdfplumber
- Frontend: HTML, CSS, JS
